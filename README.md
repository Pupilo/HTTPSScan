# HTTPSScan
Shell script for testing the SSL/TLS Protocols

Detect the SSL/TLS Vulnerabilities:

* SSLv2 (CVE-2011-1473)
* Poodle (CVE-2014-3566)
* FREAK (CVE-2015-0204)
* Weak Ciphers

Usage:

bash httpsscan.sh [target] [port]
